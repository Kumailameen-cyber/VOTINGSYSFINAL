# Online Voting System

An Online Voting System developed as a **Software Construction & Development (SCD)** course project using **ASP.NET Core 8 MVC**.

The system provides a secure and user-friendly platform for conducting online elections, allowing voters to cast votes digitally while enabling administrators to manage elections, candidates, and results efficiently.

## Technology Stack

- **Frontend:** Razor Views, HTML5, CSS3, Bootstrap
- **Backend:** ASP.NET Core 8 MVC (C#)
- **Database:** SQL Server
- **Authentication:** ASP.NET Identity
- **Version Control:** Git & GitHub
- **IDE:** Visual Studio 2022

## Key Features

### Voter
- Secure voter login
- Cast vote online
- View voting confirmation

### Candidate
-View election details
-Check nomination/participation status
-View personal vote count (read-only)

### Admin
- Manage voters and candidates
- Create and manage elections
- View election results
